Web Scraper for Fetching Exercise Data and Completing Tasks with OpenAI LLM

The project involves building a web scraper that interacts with an online exercise platform to fetch exercise data. This scraper will then send an HTTP request to an OpenAI Large Language Model (LLM) to process and respond to the exercise. Once the model returns the response, the scraper will complete the exercise accordingly.

Tools


nodejs, puppeteer library

This project aims to build a car tracking system for a company that allows for centralized data collection and performance analysis.
